# Neutronic Settings

The settings page allows users to configure the game in various ways. All the changed settings are saved automatically in the local account.

## Account Settings

The same as described in the [Account](accounts.md) section, including:

- Display name modification.
- Importing/Exporting account data.
- Deleting the account.

## Game Settings

- Enable/Disable background animations.
- Background music volume. (Not yet implemented)
- Sound effects volume. (Not yet implemented)

## Hotkey Configuration

See [Hotkeys](hotkeys.md) for details.
